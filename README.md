## 进服流程
启动游戏 -> 点击多人游戏 -> 点击添加服务器 -> 输入服务器地址并连接


## （推荐）Java 版我的世界安装教程：（ ￣▽￣）
1. 打开 pcl2
2. 点击下载
3. 点击自动安装
4. 点击“正式版”
5. 选择“1.20.4”版本
6. 开始安装
